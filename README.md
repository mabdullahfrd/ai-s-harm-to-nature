# AI's Harm to Nature

This repository contains the source code for **AI's Harm to Nature**, an educational website that explores the environmental impact of modern artificial intelligence.

The project aims to present information in a visual and accessible way, highlighting how AI systems consume electricity, water, and computational resources, and how the rapid expansion of data centers contributes to a growing environmental footprint. Rather than arguing against AI, the website encourages a more informed discussion about the environmental trade-offs that come with increasingly powerful models.

## About the Project

The website covers topics including:

* The energy required to train and run AI models
* Water consumption used for cooling data centers
* Carbon emissions associated with large-scale AI infrastructure
* The rapid growth of AI computing resources
* The relationship between technological progress and environmental sustainability

The goal is to make these topics easier to understand through an interactive web experience instead of long-form articles or research papers.

## Running Locally

Clone the repository:

```bash
git clone https://github.com/mabdullahfrd/ai-s-harm-to-nature.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Purpose

This project was built to encourage discussion around one of the less visible aspects of artificial intelligence—its environmental cost. As AI becomes more integrated into everyday life, understanding its impact on natural resources is becoming increasingly important. The website is intended as an educational resource that brings together research, data, and visual storytelling to make this topic more approachable.

## Disclaimer

The information presented on this website is intended for educational purposes. Environmental estimates surrounding AI continue to evolve as hardware, infrastructure, and model efficiency improve. Readers are encouraged to explore the referenced research and multiple sources when forming conclusions.
